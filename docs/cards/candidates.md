| **Полоса** | синяя (категория)        |


**Пример:**


## Достижение

**Факт из карьеры** — конкретная ситуация и результат. Не должность и не роль: то, что могло случиться **у любого** в IT-команде.

### Принципы формулировок


| Можно                                        | Нельзя                              |
| -------------------------------------------- | ----------------------------------- |
| «Починил прод в 3:00 — поднялся за 12 минут» | «Как CTO выстроил архитектуру»      |
| «Месяц без выходных — релиз вышел в срок»    | «Собрал команду из 12 человек»      |
| «Нашёл баг, который гоняли три месяца»       | «Привлёк $2M ARR», «Выиграл тендер» |
| «Откатил деплой за 4 минуты»                 | «Управлял отделом из 50 человек»    |
| Ситуация + результат с возможной **ценой**   | Однозначно хвалебные формулировки   |


Колонка **«Тон»** — для баланса, на карту не печатается.

### Формат карты


| На карте   | Содержание                        |
| ---------- | --------------------------------- |
| **Текст**  | ситуация + результат, до ~15 слов |
| **Полоса** | золотая (категория)               |


**Пример:**

```
Достижение

Починил прод в 3:00 ночи —
сервис поднялся за 12 минут
```

## Двусмысленные карты за столом

Правило для **любой** открытой карты кандидата (кроме бэкграунда в раунде 1), если стол спорит о знаке:

1. **30 секунд** — аргументы за плюс и минус под текущий Питч, Условия и Риск.
2. Ведущий фиксирует решение одним словом: **плюс** / **нейтрально** / **минус**.
3. Решение действует **до конца раунда**.

Примеры:

| Карта | + под контекст | − под контекст |
| ----- | -------------- | -------------- |
| «Только ночью продуктивен» | DreamDesk, тихий офис | Стендап × 15, дежурство 24/7 |
| «Собственное облако в подвале» | Runway вчера, burn облака | Compliance, пожарная инспекция |
| «Починил прод в 3:00» | Баг открыл Ад, инцидент | Вирус выгорания, команда устала |

### Балансировка руки при раздаче

- Не давать **3+ карт одного знака** при «холодном» чтении (ведущий смотрит колонку «Тон»).
- Прямая контр-пара на бэкграунд (fintech + «не трогаю регуляторику») — **редко**, для драмы.
- Не больше **2 абсурдных** и **2 провокационных** карт на все 6 категорий в одной руке.
- После раздачи модератор может поменять **1 карту**, если комбинация явно сломана.
