# «Стартап и печеньки»

Дискуссионная настольная IT-игра: попасть в core team безумного глобального проекта, убеждая других в своей ценности под конкретный питч (телепорт еды, Марс, мессенджер с умершими и т.д.). Зарплата — по ситуации.

## Быстрый старт

1. Распечатайте MVP-колоду из [playtest/mvp-deck.md](playtest/mvp-deck.md).
2. Соберите 4–6 игроков, 60–90 минут.
3. Играйте по [rules.md](rules.md).
4. Заполните фидбек и сводку в [playtest/protocol.md](playtest/protocol.md).

## Ключевые принципы

- Без должностей — только бэкграунд, навыки, достижения.
- Контекст партии (безумный Питч + условия работы + риски) определяет **знак** карты: плюс, минус или нейтрально.
- Карты — нейтральные факты; «Личный факт» не означает «минус».
- Отклонённые остаются за столом и голосуют.
- Реалистичный IT-тон и юмор в абсурдных слотах, играем персонажей.
